CSS Flexbox

Overview

Flexbox is a layout model that allows you to design a complex layout structure with a more efficient and predictable way to arrange items in a container.

Code Example

/* Flex container */
.container {
    display: flex;
    justify-content: space-between; /* Aligns items with space in between */
}

/* Flex items */
.item {
    flex: 1; /* Each item takes equal space */
    margin: 10px; /* Space around items */
}

Instructions

  1. Set up a container `
    ` with the class `container`.
  2. Add three child `
    ` elements inside it with the class `item`.
  3. Apply the provided CSS to style the container and items.
  4. Experiment by changing the `justify-content` property to `center` or `flex-end` and observe the changes.
